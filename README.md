# Automotive-Sales-Insights
End-to-end Power BI project for analyzing automotive sales performance with dynamic filters, DAX measures, KPI cards, maps, and drill-through reporting.

# Car Sales Dashboard | Power BI

📌 Project Overview

This project presents an interactive Power BI dashboard designed to analyze and monitor car sales performance across multiple dimensions including:

-Sales Performance
-Dealer Performance
-Vehicle Body Style
-Regional Sales
-Color Preferences
-Sales Trend Analysis
-Company-wise Performance

The dashboard enables business users to make data-driven decisions using dynamic filters and KPI-driven insights.
